SISTEMA DE ARCHIVOS DE USUARIOS - SGT PORTAL
==================================================

Esta carpeta contiene las carpetas personales de todos los usuarios del sistema.

ESTRUCTURA:
- Cada usuario tiene una carpeta con su DNI/NIE como nombre
- Las subcarpetas se crean automáticamente según el rol del usuario
- Los archivos .gitkeep mantienen las carpetas vacías en el control de versiones

ACCESO:
- Solo los administradores pueden acceder a todas las carpetas
- Cada usuario puede acceder únicamente a su carpeta personal
